# Feature 004/005/006 - 人工测试指导

## 功能概述

**功能名称**: 宽高比检测 + Pipeline Orchestrator + PPTX Builder
**版本**: v0.1.0
**测试日期**: [待填写]
**测试人员**: [待填写]

**功能描述**:
实现 PDF→截图→PPTX 的完整链路。包含三个紧密耦合的功能：
- **004 宽高比检测**: 检测 PDF 页面宽高比，映射到标准 PPTX 幻灯片尺寸
- **005 Pipeline Orchestrator**: 串行调度 Parser → Builder，错误隔离
- **006 PPTX Builder**: 将页面截图作为幻灯片图片写入 PPTX 文件

---

## 测试环境

### 前置条件
- Python 3.12+ 已安装
- 项目依赖已安装: `uv sync --all-extras`
- CLI 可用: `uv run noteeditor --help`

### 测试文件准备
准备以下测试 PDF 文件：
- `test_16x9.pdf`: 16:9 比例的 PDF（标准 NotebookLM 导出格式）
- `test_4x3.pdf`: 4:3 比例的 PDF
- `test_multipage.pdf`: 至少 3 页的 PDF
- `test_single.pdf`: 仅 1 页的 PDF

### 环境要求
- 操作系统: Windows 10/11
- 终端: PowerShell / CMD / Git Bash
- PPTX 查看器: Microsoft PowerPoint / WPS Office / LibreOffice Impress / Google Slides
